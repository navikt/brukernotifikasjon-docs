# brukernotifikasjon-docs
Teknisk dokumentasjon av Team personbruker sin løsning for brukernotifikasjoner

### For å kjøre lokalt:
```
$ pip install mkdocs-material
$ mkdocs serve
```
Siden skal nå vises her: http://127.0.0.1:8000
### Build
```
$ mkdocs build
```
### Deploy
```
$ mkdocs gh-deploy
```
